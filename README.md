# LibraryAPI

API REST para gerenciamento de uma livraria, construída com **Spring Boot** e **PostgreSQL**.  
Atualmente, o módulo de **Autores** está implementado com cadastro e busca com filtros dinâmicos.

## Tecnologias
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- MapStruct (mapeamento DTO ↔ Entity)
- Lombok
- Auditoria JPA (`@CreatedDate`, `@LastModifiedDate`)

---

## Como executar o projeto (local)

### Pré-requisitos
- Java 17+
- Maven
- PostgreSQL rodando localmente
